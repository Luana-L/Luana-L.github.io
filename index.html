WELCOME TO THE FLOOD SIMULATION!

The goal of this game is to reduce the damage that will be caused by the imminent flood given a certain budget. You'll create reservoirs in order to hold excess water and prevent excessive flooding.

1. You have a set budget of $10,000,000 to spend. The cost to build a reservoir is $1,000,000 per km2.

2. You can toggle the elevation overlay using the switch in the top of the tool box.

3. The flood will begin in 5 minutes, time starting once you've closed this window.

4. This start window is accessible at any time if you click on the help icon on the right!

5. Last but not least, have fun! Good luck :)

Note: Don't delete reservoirs one vertex at a time! Even if you've removed all the points, it is still stored as a reservoir in the system unless you use the delete option in the pop up. Thank you!
